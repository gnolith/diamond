# Maintainers

- [@kcsfelty](https://github.com/kcsfelty) — sole maintainer, repository owner, releases,
  security triage, schema compatibility, and final merge authority.

Security reports must use the private GitHub Security Advisory flow described
in `SECURITY.md`, not public issues or personal contact details.

The project explicitly accepts a single-maintainer bus-factor risk for the
experimental `0.1.x` series. Support is best-effort and project activity may
pause when the maintainer is unavailable. Sustained contributors may be invited
as maintainers later, but contribution does not imply maintainer status.
